## 사용 방법 (보고 루틴 ③)

1. **하루 작업이 끝나면** Task ID + 실적(%)을 알려 주세요.  
   예: `T03 90% — 홈·피드 와이어프레임 완료`
2. AI가 아래 3곳을 갱신합니다.

```
reports/YYYY-MM-DD.md   ← 오늘 스냅샷 새로 생성 (고정 보존)
daily_report.md         ← 이 대시보드 (누적 % 갱신)
WBS.md                  ← Task별 실적(%) 반영
```

### 진척률 계산

```
프로젝트 실적(%) = Σ (Task 실적(%) × 가중치) / 100
프로젝트 계획(%) = Σ (Task 계획(%) × 가중치) / 100
Gap = 프로젝트 실적(%) − 프로젝트 계획(%)
```

> Task 11개 · T01–T09 가중치 **9%** · T10–T11 가중치 **10%**
